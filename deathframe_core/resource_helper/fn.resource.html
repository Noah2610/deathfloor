pub fn resource<P>(name: P) -> PathBuf where
    P: Into<PathBuf>, 

Helper function, that returns a PathBuf to the resource file under the project's resources directory. Converts forward-slashes "/" to back-slashes "", if OS target is windows. With the debug feature, this function will also print a warning message to stderr if the file doesn't exist.
